How to run Nanvix?

Nanvix currently supports x86-based PC platforms. You can run it either in a real platform or in a real machine.

You do not need to install Nanvix in your hard drive. You can download a pre-build Live System Image, or alternatively build one from the source.

How to get involved?

Nanvix is a GPL v3 open source project. Putting it simply, this means that you can freely download the source code, hack it, and publish your changes.

If you want to stay tuned with the latest changes and the development that in ongoing, watch out our mailing list.
